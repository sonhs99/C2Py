# C2Py
The Tiny Language Interpreter

## How to Use
	Usage : C2Py <filename> <option>

<filename\> : file name (include extension)

<option\> : To show how the Interpreter works

-p :	show Parse Tree

-a :	show Abstract Syntex Tree
